# About this event

This session is perfect for anyone who wants to learn how to use GitHub, a code hosting platform for version control and collaboration. We will be covering the basics of how to create a repository, how to fork and clone repositories, and how to make changes and submit a pull request.

### Requirements:

- A laptop
- A GitHub account (sign up at https://github.com/)
